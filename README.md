# Go Fundamentals Course

Welcome to the **Go Fundamentals Course** repository by Bayt-al-Hikmah! This collection of lecture notes is designed for beginners and intermediate learners aiming to master Go (Golang) programming fundamentals. Whether you're new to coding or refreshing your skills, these notes provide structured, step-by-step guidance through core concepts.

Bayt-al-Hikmah (House of Wisdom) is dedicated to preserving and disseminating knowledge, inspired by the historic intellectual center of the Islamic Golden Age. These lectures emphasize clear explanations, practical examples, and hands-on exercises to build a strong foundation in the Go language.

## Prerequisites

- Basic familiarity with computers and file management.
    
- No prior programming experience required (though helpful).
    
- Go toolchain and an IDE like VS Code (with the Go extension) or GoLand.

## Course Overview

This 5-lecture series covers essential Go topics progressively. Each lecture includes:

- Theoretical explanations.
    
- Code examples and snippets.
    
- Exercises for practice.
    
- References for further reading.
    

|**Lecture**|**Topic**|**Key Concepts Covered**|
|---|---|---|
|Lecture 1|Go Basics and Computer Fundamentals|How computers work, intro to programming, Go syntax, variables, user I/O with `fmt` package.|
|Lecture 2|Data Structures and Control Flow|Strings, arrays, slices, maps, comparison/logical operators, `if/else`, `switch`, `for` loops.|
|Lecture 3|Functions, Packages, and Modules|Defining functions, return values, creating packages, managing dependencies with Go Modules.|
|Lecture 4|Pointers, Structs, Interfaces, and Error Handling|Pointers, `struct` types, methods, `interface` implementation, polymorphism, `iota`, `error` handling.|
|Lecture 5|Concurrency, File I/O, and CLI Tools|Goroutines, channels, file I/O (`os` package), command-line arguments (`flag` package).|

**Recommended Pace**: 1-2 lectures per week, with time for coding practice. Total estimated time: 15-25 hours.
